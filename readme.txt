OpeningTimes.in plugin makes your customers life better through easy and simple access to opening times information.

== Description ==

OpeningTimes.in plugin is effective and convenient way to know if branch or shop is open before visiting. In addition to working hours information, other details like phone numbers, address, holidays, nearby branches would be configured and easily customized based on the requirements. There is no limit to the number of Branches. For limited branches, branch can be located with help of list. In addition, it has powerful text base search engine to search branches based on the area.

This plugin provides a service interface to www.openingtimes.in server to fetch the Branch and Opening times data. To use this plugin it's required to create and an account with the www.openingtimes.in and setup the Branches and respective opening times. www.openingtimes.in will provide Client ID and API key that is used while configuring this Plugin on the Wordpress. 

== Installation ==

This section describes how to install the plugin and get it working.
e.g.

1. Install and Activate the plugin through the 'Plugins' menu in WordPress.
2. Create an account with www.openingtimes.in
3. Add branches and set their opening times.
4. Get Client ID and API Key from the Dashboard on www.openingtimes.in
5. Find the OpeningTimes plugin setting page in the WordPress Menu on the left side of your browser window.
6. Set the Client ID, API Key and choose Theme and save it.
